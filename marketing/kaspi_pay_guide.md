# Инструкция по подключению Kaspi Pay
# Буллинг Стоп КЗ — монетизация документов 2+3

---

## Что нужно подключить
Когда готовы запустить платную версию (Документы 2+3 = 2 990 тг),
нужно подключить один из двух вариантов:

---

## ВАРИАНТ А — Kaspi Pay (рекомендуется для РК)

### Шаг 1: Зарегистрироваться как бизнес в Kaspi
1. Перейдите: **kaspi.kz/business** → "Стать партнёром"
2. Нужно: ИП или ТОО (у вас есть ADDASTRA — подходит)
3. Документы: свидетельство о регистрации, БИН, реквизиты счёта

### Шаг 2: Получить Kaspi Pay API
После одобрения вы получите:
- `KASPI_MERCHANT_ID` — ваш ID магазина
- `KASPI_API_KEY` — секретный ключ

### Шаг 3: Добавить в Railway
В Railway → Variables добавьте:
```
KASPI_MERCHANT_ID=ваш_merchant_id
KASPI_API_KEY=ваш_api_key
PAYMENT_ENABLED=true
```

### Шаг 4: Включить оплату
В Railway переменная `PAYMENT_ENABLED=true` — автоматически
включает платный режим. Документ 1 остаётся бесплатным.

---

## ВАРИАНТ Б — Простая схема (без API, сейчас)

Пока Kaspi Pay не подключён, можно принимать оплату вручную:

1. Пользователь нажимает "Оплатить" → бот отправляет его в поддержку
2. Вы отправляете номер Kaspi для перевода
3. Пользователь переводит 2 990 тг → присылает скриншот
4. Вы вручную отправляете Документы 2+3

Это уже настроено в боте — кнопка ведёт на @addastra (ваш контакт).

## Финансовый прогноз

| Сценарий | Пользователей/мес | Конверсия в оплату | Доход |
|---|---|---|---|
| Минимум | 50 | 20% = 10 чел | 29 900 тг |
| Реалистично | 200 | 25% = 50 чел | 149 500 тг |
| Хорошо | 500 | 30% = 150 чел | 448 500 тг |

Плюс заявки на консультацию ADDASTRA — отдельный доход.

## Альтернативы если Kaspi долго
- **CloudPayments KZ** — быстрее подключается, есть Telegram API
- **PayBox.money** — работает в РК, есть готовая библиотека Python
- **Stripe** — международный, но сложнее для РК резидентов
